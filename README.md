# Java Mini Projects

This repository contains beginner-friendly Java mini projects developed as part of self-learning.

## Projects Included

### 1. Student Management System
- Console-based Java application
- Uses Core Java, OOP, ArrayList, and exception handling
- Features: Add, view, search, and delete student records

### 2. Bank Account Management System
- Java console application
- Implements OOP concepts like encapsulation
- Features: Deposit, withdraw, balance check

## Technologies Used
- Java (Core Java)
- OOP Concepts
- Collections
- Console-based input/output

## Purpose
These projects were created to strengthen Core Java and Object-Oriented Programming skills.
